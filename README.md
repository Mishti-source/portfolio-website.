# Minimalist Developer Portfolio Website

A highly polished, performance-optimized personal portfolio website built entirely from scratch for the Code-A-Nova Frontend Developer Intern Assignment. The design emphasizes premium typography, intentional whitespace, a sophisticated editorial layout, and custom light/dark color schemas.

## 🚀 Live Deployment
👉 **https://github.com/Mishti-source/portfolio-website..git**

---

## 🛠️ Built With (Tech Stack)
- **Markup:** HTML5 (Strict semantic elements like `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` used for high accessibility mapping)
- **Styling:** Tailwind CSS (Utility-first configuration for fluid responsive layouts)
- **Scripting:** Vanilla JavaScript (ES6+ native asynchronous engines for interactivity)
- **Typography:** Google Fonts (Playfair Display & Plus Jakarta Sans)
- **Iconography:** Font Awesome Web Fonts
